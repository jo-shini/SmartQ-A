# Smart Q&A API (RAG System)
 
## 🚀 Overview
Backend API using Node.js + Express + MongoDB + LLM (Groq)
 
Implements Retrieval-Augmented Generation (RAG) with:
- Document retrieval
- Context grounding
- Structured LLM response
